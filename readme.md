## 超级音乐。
### 您可以播放：QQ会员的歌曲、即使是QQ会员也需要单独购买的歌曲、QQ网页端没有的高品质歌曲、QQ因版权问题全平台都不能播放的歌曲。
### 安装后请您以下操作：
1. 进入QQ歌单、专辑、歌手、歌曲页面，点击浏览器右上角的本插件图标，在弹出的播放器界面播放歌曲。如果当前页并不是以上页面，会播放最近一次的历史记录。如果没有历史记录，超级音乐为您推荐了一个默认歌单（此歌单非本人制作，亦非本人喜好，纯属随机，欢迎您推荐优秀的歌单）。
### 目前仅支持QQ音乐，后期可能加入其他歌源。本人不支持任何盗版行为，同时希望您支持正版歌曲。本插件旨在增加官方尚不支持的功能或优化播放体验。
